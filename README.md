# ARQUEOB

Sistema profesional de arqueo, conduces y cuentas por cobrar hecho con Flask, PostgreSQL, SQLAlchemy, Flask-Migrate y Bootstrap 5.

## Módulos

- Dashboard con indicadores de cartera y cobros.
- Administración de clientes, búsqueda y facturas múltiples.
- Cobros aplicados a una o varias facturas, con historial y recibo PDF.
- Arqueo de caja y reporte PDF.
- Conduce de envío y PDF de dos copias.

## Requisitos y arranque

La aplicación necesita PostgreSQL; no usa SQLite. Consulte [MIGRACION.md](MIGRACION.md) para configurar `.env`, ejecutar migraciones y arrancar el servidor.
